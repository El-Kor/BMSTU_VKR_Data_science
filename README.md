Выпускная квалификационная работа по курсу "Data Science" МГТУ им. Н. Э. Баумана на тему "Прогнозирование конечных свойств композиционных материалов".
Тема: Прогнозирование конечных свойств композиционных материалов.

В процессе работы были выполнены следующие этапы и получены ре-зультаты: 
1. Анализ и предобработка данных. Проведен разведочный анализ данных датасета. Были устранены пропуски, удалены аномальные значения (выбросы) и выполнена нормализация признаков (Min-Max Scaling). 
2. Исследование методов машинного обучения. В работе были применены и сопоставлены различные подходы: линейная регрессия (включая Lasso), метод опорных векторов (SVR), метод k-ближайших соседей (k-NN) и метод «Случайный лес» (Random Forest). 
3. Разработка нейронной сети. С помощью библиотеки TensorFlow/Keras была спроектирована нейросеть. Модель была ориентирована на прогноз соотношения «матрица-наполнитель». 
4. Программная реализация. Разработано веб-приложение на базе фрейм-ворка Flask. Интерфейс приложения позволяет конечному пользователю вводить технологические параметры (плотность, модули упругости, углы нашивки и др.) и получать прогноз соотношения матрица-наполнитель.
